This repository contains **multiple Java and Spring Boot projects** covering advanced topics such as **Unit Testing, Spring Security with JWT, Hibernate JPA, MongoDB, Microservices, Caching (Caffeine), and Artificial Intelligence (Gemini)**.  

All projects are structured for **modularity, clean architecture, and production-ready best practices**.
